# Window Replacements of Plano

Premium window replacement website for Window Replacements of Plano, TX. Authorized dealer for Andersen, Pella, and Jeld-Wen windows.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: CSS Modules
- **Typography**: Cormorant Garamond & Outfit (Google Fonts)
- **Language**: TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Project Structure

```
src/
├── app/
│   ├── globals.css    # Global styles & CSS variables
│   ├── layout.tsx     # Root layout with SEO metadata

├── components/

│   ├── hero-slider.tsx    # Full-screen image slider
│   ├── content-sections.tsx   # All page content sections
│   └── footer.tsx     # Footer components
```
